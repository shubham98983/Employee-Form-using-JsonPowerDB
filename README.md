# Employee-Form-using-JsonPowerDB
# Description
This project is a web-based Student Registration Form built using HTML, Bootstrap, JavaScript, and the JPDB (Login2Explore) API. It supports adding and updating student records with proper form validation and controlled button behavior based on the existence of the primary key (Roll-No).

# Benefits of using JsonPowerDB
- Simple and schema-free database that stores data in JSON format, making it easy to use with web applications
- High performance with fast read/write operations and minimal server-side processing
- Built-in REST APIs reduce backend development effort
- Ideal for small to medium projects, rapid prototyping, and learning database concepts
- Secure access using connection tokens and role-based permissions
